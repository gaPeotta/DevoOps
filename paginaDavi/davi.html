O que é Pull Request?

Pull Request (PR) é uma funcionalidade do GitHub utilizada para solicitar que alterações feitas em uma branch sejam analisadas e adicionadas à branch principal do projeto. Ele permite que outros desenvolvedores revisem o código antes da integração definitiva.

Esse processo é muito importante no desenvolvimento colaborativo, pois ajuda a evitar erros, melhorar a qualidade do código e manter o histórico das modificações organizado.

Resumo: você cria uma branch, faz alterações, envia para o GitHub e abre um Pull Request para que suas mudanças sejam revisadas e aprovadas.
